# Access Control Lab — Suggestions & Community

Community hub for **[Access Range](https://access-range.vercel.app)** — 21 hands-on
labs on broken access control, IDOR, JWT, CSRF, CORS, business logic and account takeover.

This repo holds **no lab source code**. It exists only for:

- 🐛 **Bug reports** — something broken on the platform? [Open an issue](../../issues/new).
- 💡 **Lab suggestions / feature requests** — ideas for new labs or improvements.
- 🔐 **Security disclosures** — found a real vuln on the platform (outside the labs)?
  [Report privately via a security advisory](../../security/advisories/new).
  Full policy: <https://access-range.vercel.app/security>.

The labs are intentionally vulnerable. The platform itself is not — please keep
disclosures scoped to the platform shell, not the `/labs` curriculum.
